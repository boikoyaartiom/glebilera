# 📧 Настройка отправки анкеты на Email (Web3Forms)

## Почему Web3Forms?
- ✅ **Бесплатно** до 250 писем в месяц
- ✅ **Без верификации** Google
- ✅ **2 минуты** на настройку
- ✅ Ответы гостей приходят на ваш email

---

## Пошаговая инструкция

### Шаг 1: Получите API ключ
1. Перейдите на **https://web3forms.com**
2. Введите ваш email (куда будут приходить ответы)
3. Нажмите **Create Access Key**
4. Проверьте почту — вам пришёл код подтверждения
5. Подтвердите и скопируйте **Access Key**

### Шаг 2: Вставьте ключ в проект
Откройте файл `src/config.ts` и замените:

```typescript
// Замените "ВАШ_КЛЮЧ" на ваш реальный ключ
export const WEB3FORMS_ACCESS_KEY = "abc123def456..."; 

// Измените способ отправки на "email"
export const DELIVERY_METHOD = "email";
```

### Шаг 3: Пересоберите проект
Нажмите кнопку сборки или выполните:
```bash
npm run build
```

### Шаг 4: Загрузите на хостинг
Загрузите папку `dist` на Netlify/Vercel

### Готово! 🎉
Теперь каждый ответ из формы будет приходить вам на email!

## Как выглядит письмо

Вы получите письмо с такой темой:
```
RSVP: Иван Иванов — Придёт
```

И содержимым:
```
Гость: Иван Иванов
Статус: Придёт
Количество гостей: 2
Напитки: Красное вино, Шампанское
Трансфер: Нужен
Пожелания: Будем рады разделить этот день с вами!
```

## Если хотите и Google Таблицу тоже

Можно настроить两者:
1. **Email** — для мгновенных уведомлений
2. **Google Forms** — для сбора в таблицу

Создайте Google Form с такими вопросами:
- Ваше имя
- Статус (Придёт/Не смогу)
- Количество гостей
- Напитки (множественный выбор)
- Трансфер
- Пожелания

Затем свяжите форму с Google Таблицей (связь создаётся автоматически).
