## Пошаговая инструкция

### Шаг 1: Откройте Apps Script
1. Откройте вашу таблицу по ссылке выше
2. В меню нажмите **Extensions** → **Apps Script**

### Шаг 2: Вставьте код
1. Удалите весь стандартный код в редакторе
2. Скопируйте код из файла `GOOGLE_SCRIPTS_CODE.txt`
3. Вставьте в редактор Apps Script
4. Нажмите **💾 Save** (или Ctrl+S)

### Шаг 3: Создайте заголовки
1. В верхней панели выберите функцию **setupHeaders**
2. Нажмите **▶️ Run**
3. Разрешите доступ (если попросит)
4. Заголовки появятся в таблице!

### Шаг 4: Опубликуйте как веб-приложение
1. Нажмите **Deploy** → **New deployment**
2. Нажмите на ⚙️ (шестерёнку) → **New app**
3. Настройте:
   - **Type:** Web app
   - **Description:** Wedding RSVP
   - **Execute as:** Me
   - **Who has access:** Anyone
4. Нажмите **Deploy**
5. **Скопируйте URL** (выглядит как `https://script.google.com/macros/s/ABC123.../exec`)

### Шаг 5: Вставьте URL в проект
Откройте файл `src/config.ts` и замените:

```typescript
export const GOOGLE_SHEETS_URL = "https://script.google.com/macros/s/ВАШ_СКОПИРОВАННЫЙ_URL/exec";
```

### Шаг 6: Пересоберите проект
Нажмите кнопку сборки и загрузите папку `dist` на хостинг.

## 🔍 Как проверить, что всё работает?
1. Откройте форму на сайте
2. Заполните и отправьте тестовый ответ
3. Обновите Google Таблицу
4. Должна появиться новая строка с данными!

---

## ⚠️ Если не работает
- Убедитесь, что URL скопирован правильно
- Проверьте, что скрипт опубликован (Deploy → Manage deployments)
- Откройте таблицу в режиме просмотра, а не редактирования
